# Sales Expense Calculator

A colorful, interactive web app for tracking and calculating sales expenses per salesperson.

## Features

- **Salesperson Selection** — Switch between salespeople to manage individual expenses
- **Expense Toggles** — Enable/disable categories (Travel, Meals, Lodging, Phone, Flights, Gifts, Fuel, Parking) with dollar amounts
- **Live Totals** — Real-time expense summary with active category count
- **Reset & Export** — One-click reset and copy-to-clipboard summary

## Tech Stack

- **React 18** + TypeScript
- **Vite 5** — Fast dev server & build
- **Tailwind CSS v3** — Utility-first styling with custom design tokens
- **shadcn/ui** — Accessible UI primitives
- **Lucide React** — Icon library

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Design System

- **Fonts:** Space Grotesk (headings), DM Sans (body)
- **Primary:** Purple (`hsl(250, 65%, 50%)`)
- **Secondary:** Pink (`hsl(340, 75%, 55%)`)
- **Accent:** Teal (`hsl(170, 65%, 40%)`)
